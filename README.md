# Instrymentalka

Практика №7 Динисаламова Ильяса и Мингазова Альмира, Ис-201кН
 
Для использования требуется выполнить следующие действия:
 
1. Сначала установить форк Flask:

```cmd
pip install flask
```

2. Перейти в директорию "Task1":
```cmd
cd MvcDataViews
```

3. Запустить код:
```cmd
python app.py
```
